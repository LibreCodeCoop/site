# Desenvolvimento de apps

O LibreSign é um app para Nextcloud. Você pode criar outro app para demonstrar seus conhecimentos.

Sugestões:

* Um jogo simples para rodar no navegador e demonstrar uma boa base técnica
* Uma integração com outra aplicação
  * [Kimai](https://www.kimai.org/) exibindo as últimas atividades registradas. Requer configuração administrativa para informar os dados de acesso da instância utilizada. Documentação [aqui](https://docs.nextcloud.com/server/latest/developer_manual/digging_deeper/dashboard.html).
  * LimeSurvey, com um dashboard listando surveys existentes. Requer configuração administrativa para informar os dados de acesso da instância utilizada. Documentação [aqui](https://docs.nextcloud.com/server/latest/developer_manual/digging_deeper/dashboard.html).
* Atualizar um app existente que esteja abandonado. Exemplos:
  * [Player de rádio web](https://apps.nextcloud.com/apps/radio)
